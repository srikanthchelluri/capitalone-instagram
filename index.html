Analyze #CapitalOne Instagram Data

Srikanth Chelluri, University of Virginia



This is the webpage for my solution to the Capital One Summit for Software Engineers' challenge problem.
The challenge problem can be found on MindSumo. I use the Instagram API to gather and analyze data.

The front-end uses Twitter's Bootstrap. The API calls are requested via JQuery (AJAX) and the logic is JavaScript.
Further details were submitted via MindSumo (or can be requested via email).
N.B. The backend was built with ease-of-readability in mind, not security.


Part of my application requires an access_token from Instagram's API.
For it to work, you'll need to grant my application permission (link below).
The application won't post to Instagram or otherwise act on your behalf.

Authenticate this application.
